# 如何使用 vue 文件

## 简介
在初学 Vue 时可能需要预览 Vue 单文件组件（Single-File Component，缩写为 SFC）。它是一种可复用的代码组织形式，将从属于同一个组件的 HTML、CSS 和 JavaScript 封装在使用`.vue`后缀的文件中。
